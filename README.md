# Homebrew Tap for ChemEx

This tap provides [ChemEx](https://github.com/gbouvignies/ChemEx), an NMR Chemical Exchange Analysis Tool.

## Installation

```bash
# Add this tap
brew tap gbouvignies/chemex

# Install ChemEx
brew install chemex
```

## About ChemEx

ChemEx is an advanced, open-source software specifically designed for analyzing NMR experimental data to characterize chemical exchange processes. It's ideal for researchers and scientists in biochemistry and molecular biology, particularly for analyzing CPMG relaxation dispersion and CEST experiments.

## Documentation

For more information, visit the [ChemEx homepage](http://gbouvignies.github.io/ChemEx/).
